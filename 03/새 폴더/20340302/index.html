이틀전까지 한내용 태그: div p h1~h6 span br img table a 스타일을 줘보자 속성(attribute)style을 주자
style 속성 = css
width : 너비, max-width, min- width
height : 높이, max height min height
background-color : 배경색, 색상을 태그를 입힐수있다. rgb(255,255,255);
padding(태그 내부 여백), margin(외부의 여백): top, right bottom,left(순서를 뜻한다.)
margin의 경우 값을 하나만 썼을때는 그 값이 사방면에 다적용이 된다. 사방면을 각각 주고싶을때는 top right bottom left 모두 작성해줘도 된다.
두개씩 짝짓는것도 가능한데 그경우는 상하 , 좌우 두가지 경우에 해당한다.
box-sizing : 박스의 크기를 어떻게 설정할지를 기준을 설정
속성을 추가 할때 속성 명 = "속성 값" display : 영역의 노출크기를 어떻게 설정할지 기준을설정한다. 부모태그를 기준으로 모든 너비를 차지한다. 높이는 자식의 기준으로 부모의 영역만큼만 차지한다. 블럭일 경우.
