# 브런치 영역을 나눠보자

# 브런치는 저장소의 작업공간

- master는 최종 작업물 다른 브런치를 만들어서 여럿이서 작업을 하거나 혼자
    작업할때 작업의 내용을 나눠서 작업하고 최종 작업물로 병합한다.

    master -> dev -> dev-> dev-> master로 최종결과물 병합.
